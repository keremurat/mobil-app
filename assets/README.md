# Assets

Bu klasör uygulama için gerekli asset dosyalarını içerir.

## Gerekli Dosyalar

Expo uygulamanızı başlatmadan önce aşağıdaki dosyaları eklemeniz önerilir:

- `icon.png` - 1024x1024 px (uygulama ikonu)
- `splash.png` - 1284x2778 px (splash screen)
- `adaptive-icon.png` - 1024x1024 px (Android adaptive icon)
- `favicon.png` - 48x48 px (web favicon)

Şu an için uygulama bu dosyalar olmadan da çalışacaktır, ancak Expo varsayılan iconları kullanılacaktır.
